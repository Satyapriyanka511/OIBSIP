# Task 4: Research Report on Common Network Security Threats

## Objective

The objective of this task is to research and understand some of the most common network security threats that affect computer networks and online systems. This report explains how these threats work, the risks they create, and the possible prevention techniques that can be used to protect networks.

## Introduction

In the modern digital world, computer networks are used everywhere for communication, business operations, online transactions, and data storage. Because of this, networks have become a major target for cyber attackers. Network security threats are malicious activities designed to damage systems, steal sensitive information, or disrupt normal operations of a network.

Cyber attackers use different techniques to exploit vulnerabilities in systems. Some of the most common network attacks include Denial-of-Service (DoS) attacks, Man-in-the-Middle (MITM) attacks, and Spoofing attacks. Understanding these threats is very important for maintaining the security and reliability of computer networks.

## Denial-of-Service (DoS) Attack

A Denial-of-Service (DoS) attack occurs when an attacker sends a large amount of traffic to a server or network in order to overload it. When the server receives more requests than it can handle, it becomes slow or completely unavailable to legitimate users.

For example, if thousands of fake requests are sent to a website at the same time, the server may crash or stop responding. As a result, real users will not be able to access the website or service.

### Impact of DoS Attack

* Websites or online services may become unavailable.
* Business operations can be disrupted.
* Organizations may suffer financial losses and reputational damage.

### Prevention Methods

* Using firewalls and network security devices.
* Implementing traffic filtering and rate limiting.
* Using Intrusion Detection Systems (IDS) and Intrusion Prevention Systems (IPS).

## Man-in-the-Middle (MITM) Attack

A Man-in-the-Middle attack happens when an attacker secretly intercepts communication between two parties. The attacker places themselves between the sender and the receiver and can monitor, modify, or steal the transmitted data.

For example, if a user connects to a public Wi-Fi network and logs into a banking website, an attacker may intercept the communication and capture sensitive information such as login credentials or financial details.

### Impact of MITM Attack

* Sensitive information can be stolen.
* Login credentials may be compromised.
* Financial fraud or identity theft can occur.

### Prevention Methods

* Using HTTPS and secure encryption protocols.
* Avoiding public or unsecured Wi-Fi networks.
* Using Virtual Private Networks (VPNs) for secure communication.
* Implementing strong authentication methods.

## Spoofing Attack

Spoofing is a type of attack where an attacker disguises themselves as a trusted device or user in order to gain unauthorized access to a network. In spoofing attacks, the attacker manipulates identity information to appear legitimate.

### Types of Spoofing

**IP Spoofing:**
The attacker sends network packets with a fake IP address to hide their identity or impersonate another system.

**Email Spoofing:**
Attackers send emails that appear to come from trusted sources in order to trick users into revealing sensitive information.

**ARP Spoofing:**
In this attack, the attacker sends fake Address Resolution Protocol (ARP) messages to link their MAC address with the IP address of another device on the network.

### Impact of Spoofing

* Unauthorized access to systems and networks.
* Data interception and manipulation.
* Malware distribution and phishing attacks.

### Prevention Methods

* Implementing strong authentication mechanisms.
* Monitoring network traffic for suspicious activity.
* Using packet filtering and secure network configurations.

## Importance of Network Security

Network security is essential for protecting sensitive information and maintaining the integrity of systems. Organizations must adopt proper security practices such as encryption, monitoring tools, firewalls, and regular security audits to protect their networks from cyber threats.

Security awareness among users is also very important. Educating employees and users about cyber threats can significantly reduce the risk of successful attacks.

## Conclusion

Network security threats are constantly evolving as attackers develop new techniques to exploit vulnerabilities. Attacks such as Denial-of-Service, Man-in-the-Middle, and Spoofing can cause serious damage to systems and organizations if proper security measures are not implemented. By understanding these threats and applying appropriate prevention methods, organizations can improve their overall network security and protect their valuable data.
